# dompet-harian

Aplikasi Catatan Pengeluaran Keuangan Harian menggunakan Ruby (Sinatra), PostgreSQL, dan Vanilla CSS.

## Fitur Utama
- **Layout Responsif**: Desktop Sidebar & Mobile Bottom Navigation Bar (dengan Glassmorphism style).
- **Penginputan Cepat**: Drawer modal slide-up yang interaktif untuk mobile & overlay modal untuk desktop.
- **Rupiah Formatter**: Format angka otomatis ke Rupiah saat mengetik nominal.
- **Analisis & Laporan**: Visualisasi porsi kategori (Doughnut Chart) & tren pengeluaran harian (Line Chart) menggunakan Chart.js.
- **Migrasi DB Otomatis**: Secara otomatis memverifikasi dan membangun tabel PostgreSQL saat aplikasi dijalankan.
